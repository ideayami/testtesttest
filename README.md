# 東方Project風シューティングゲーム

本格的な弾幕シューティングゲーム（東方Projectインスパイア）

## 🎮 特徴

- 複雑な弾幕パターンシステム
- スペルカードシステム
- ボム・パワーアップシステム
- 複数ステージ・ボス戦
- ブラウザで動作

## 🚀 クイックスタート

```bash
# 依存関係のインストール
npm install

# 開発サーバー起動
npm run dev

# ビルド
npm run build

# テスト実行
npm test
```

## 📁 プロジェクト構造

```
src/
├── scenes/          # ゲームシーン
├── entities/        # ゲームオブジェクト
├── systems/         # ゲームシステム
├── patterns/        # 弾幕パターン
├── config/          # 設定ファイル
└── assets/          # リソース
```
